
# AutoCurriculum RL Engine — Self‑Learning Math Curriculum Generator

AutoCurriculum RL Engine is an intelligent reinforcement‑learning framework that designs personalized math curricula in real time. Unlike static adaptive systems, AutoCurriculum continuously experiments with exercise sequences, evaluates student responses, and learns which paths maximize improvement, accuracy, and engagement.

---

## 🚀 Key Features
- **Reinforcement Learning Core**  
  Learns optimal exercise sequences using reward signals from student performance.

- **Dynamic Difficulty Scaling**  
  Adjusts problem complexity based on accuracy trends and response speed.

- **Curriculum Optimization**  
  Chooses problem types (algebra, geometry, probability, sequences) to target weak skills.

- **Student Simulation Environment**  
  Includes a configurable model of student behavior for testing RL strategies.

- **Analytics & Logging**  
  Tracks accuracy, error patterns, and curriculum efficiency over time.

---

## 🧠 How It Works
1. Define a set of math exercise templates with tags for difficulty and skill type.  
2. The RL agent selects an exercise based on the current student state.  
3. The student model or real user attempts the problem.  
4. The system computes a reward (accuracy, speed, error reduction).  
5. The agent updates its policy and refines future curriculum choices.  
6. Over time, the engine converges toward optimal learning paths.
